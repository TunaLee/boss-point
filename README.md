# 흉&베라 포인트 입찰 메뉴얼

Google Sheets 데이터를 기반으로 길드 포인트 입찰 정보를 보여주는 웹 페이지입니다.

## 시작하기

### 1. Google 스프레드시트 생성

새 Google 스프레드시트를 만들고, 아래 **7개의 시트(탭)**를 정확한 이름으로 생성합니다.

| 시트 이름 | 용도 |
|-----------|------|
| `설정` | 마감일, 환율, 갱신일 등 기본 설정 |
| `현재입찰` | 현재 진행 중인 입찰 목록 |
| `즉시구매` | 재고가 있어 바로 구매 가능한 아이템 |
| `상시판매` | 항상 판매 중인 아이템 |
| `공개입찰` | 공개 입찰 아이템 |
| `블라인드입찰` | 블라인드 입찰 아이템 |
| `순번제` | 순번 기반 획득 아이템 |

### 2. 스프레드시트 공개 설정

**공유** > **링크가 있는 모든 사용자** > **뷰어** 권한으로 설정합니다.

### 3. 시트 ID 연결

스프레드시트 URL에서 시트 ID를 복사합니다.

```
https://docs.google.com/spreadsheets/d/여기가_시트_ID/edit
```

`index.html` 파일의 아래 부분에 해당 ID를 입력합니다.

```javascript
const SHEET_ID = '여기에_시트_ID_붙여넣기';
```

## 시트별 작성법

> **공통 규칙**
> - 1행은 헤더(제목)로 사용하세요. 데이터는 **2행부터** 입력합니다.
> - A열(아이템명)이 비어있는 행은 자동으로 무시됩니다.

---

### 설정

기본 설정값을 입력하는 시트입니다. **1행에 헤더 없이 바로 값을 입력**합니다.

| 열 | 내용 | 예시 |
|----|------|------|
| A | 입찰 마감일 | `3월 10일 (월) 자정` |
| B | 포인트 환율 | `1포인트 = 100원` |
| C | 갱신일 | `2026-03-04` |

**예시:**

| A | B | C |
|---|---|---|
| 3월 10일 (월) 자정 | 1포인트 = 100원 | 2026-03-04 |

---

### 현재입찰

현재 진행 중인 입찰 아이템을 등록합니다.

| 열 | 내용 | 설명 |
|----|------|------|
| A | 아이템명 | 아이템 이름 |
| B | 수량 | 입찰 가능 수량 |
| C | 포인트 | 포인트 |
| D | 가격 | 판매 가격 |
| E | 미당첨 | 낙찰 실패 시 포인트 |
| F | 비고 | 추가 메모 |

**예시:**

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| 빛나는 장갑 | 1 | 500 | 50000 | 50 | 주간 보스 |
| 전설 투구 | 2 | 1000 | 외판 | 100 | |

---

### 즉시구매

재고가 있어 바로 구매 가능한 아이템입니다.

| 열 | 내용 | 설명 |
|----|------|------|
| A | 아이템명 | 아이템 이름 |
| B | 수량 | 재고 수량 |
| C | 포인트 | 구매에 필요한 포인트 |
| D | 비고 | 추가 메모 |

**예시:**

| A | B | C | D |
|---|---|---|---|
| 경험치 물약 | 10 | 100 | |
| 귀환 주문서 | 5 | 50 | 이벤트 한정 |

---

### 상시판매 / 공개입찰 / 블라인드입찰

이 세 시트는 동일한 구조를 사용합니다.

| 열 | 내용 | 설명 |
|----|------|------|
| A | 아이템명 | 아이템 이름 |
| B | 포인트 | 포인트 |
| C | 가격 | 판매 가격 |
| D | 비고 | 추가 메모 |

#### 특수 기능: 그룹 헤더

B~C열을 모두 비워두고 A열에만 텍스트를 입력하면 **카테고리 구분선(그룹 헤더)**으로 표시됩니다.

#### 특수 가격 표기

| 가격 값 | 표시 | 의미 |
|---------|------|------|
| `0` 또는 `0원` | 초록색 텍스트 | 무료 |
| `외판` | 빨간 이탤릭 텍스트 | 외부 판매 |

**예시:**

| A | B | C | D |
|---|---|---|---|
| **주간 보스** | | | |
| 빛나는 장갑 | 500 | 50000 | |
| 전설 투구 | 1000 | 외판 | |
| **일일 보스** | | | |
| 경험치 물약 | 100 | 0원 | |

> 위 예시에서 "주간 보스"와 "일일 보스" 행은 B~C가 비어있으므로 그룹 헤더로 렌더링됩니다.

---

### 순번제

순번 기반으로 획득하는 아이템입니다.

| 열 | 내용 | 설명 |
|----|------|------|
| A | 아이템명 | 아이템 이름 |
| B | 획득 방식 | 방식 설명 |
| C | 가격 | 가격 또는 비용 |

**예시:**

| A | B | C |
|---|---|---|
| 전설 무기 | 포인트 순 | 2000 |
| 희귀 방패 | 가입 순 | 500 |

---

## 사용법

1. 위 가이드에 따라 Google 스프레드시트를 작성합니다.
2. `index.html`을 브라우저에서 열면 스프레드시트 데이터가 자동으로 로드됩니다.
3. 스프레드시트 내용을 수정한 후 **페이지를 새로고침**하면 변경사항이 반영됩니다.

## 참고사항

- 별도 서버 없이 `index.html` 파일을 직접 열어서 사용할 수 있습니다.
- 데이터 로드에 실패하면 10초 후 에러 메시지가 표시됩니다.
- 스프레드시트가 **공개(뷰어)** 상태여야 데이터를 불러올 수 있습니다.
