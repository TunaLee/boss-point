# 흉&베라 포인트 입찰 메뉴얼

Google Sheets 데이터를 기반으로 길드 포인트 입찰 정보를 보여주는 웹 페이지입니다.

## 시작하기

### 1. Google 스프레드시트 생성

새 Google 스프레드시트를 만들고, 아래 **4개의 시트(탭)**를 정확한 이름으로 생성합니다.

| 시트 이름 | 용도 |
|-----------|------|
| `설정` | 갱신일 등 기본 설정 |
| `규칙` | 포인트 규칙, 보스 드랍 등 |
| `소모품` | 소모품 아이템 |
| `길드템` | 스킬북, 장비, 길드전 보상 등 |

> `setup-sheet.gs`를 Apps Script에 붙여넣고 실행하면 자동으로 시트와 데이터가 생성됩니다.

### 2. 스프레드시트 공개 설정

**공유** > **링크가 있는 모든 사용자** > **뷰어** 권한으로 설정합니다.

### 3. 시트 ID 연결

스프레드시트 URL에서 시트 ID를 복사합니다.

```
https://docs.google.com/spreadsheets/d/여기가_시트_ID/edit
```

`index.html` 파일의 아래 부분에 해당 ID를 입력합니다.

```javascript
const SHEET_ID = '여기에_시트_ID_붙여넣기';
```

## 시트별 작성법

> **공통 규칙**
> - 1행은 헤더(제목)로 사용하세요. 데이터는 **2행부터** 입력합니다.
> - A열이 비어있는 행은 자동으로 무시됩니다.

---

### 설정

기본 설정값을 입력하는 시트입니다. **1행에 헤더 없이 바로 값을 입력**합니다.

| 열 | 내용 | 예시 |
|----|------|------|
| A | (미사용) | |
| B | (미사용) | |
| C | 갱신일 | `2026-03-04` |

---

### 소모품 / 길드템

두 시트는 동일한 구조를 사용합니다.

| 열 | 내용 | 설명 |
|----|------|------|
| A | 아이템명 | 아이템 이름 |
| B | 포인트 | 필요 포인트 |
| C | 가격 | 판매 가격 |
| D | 비고 | 추가 메모 |

#### 그룹 헤더

B~C열을 모두 비워두고 A열에만 텍스트를 입력하면 **카테고리 구분선**으로 표시됩니다.

#### 특수 가격 표기

| 가격 값 | 표시 | 의미 |
|---------|------|------|
| `0` 또는 `0원` | 초록색 텍스트 | 무료 |
| `외판` | 빨간 이탤릭 텍스트 | 외부 판매 |

---

### 규칙

포인트 규칙, 입찰 방식, 보스별 드랍 정보를 관리합니다.

| 열 | 내용 | 설명 |
|----|------|------|
| A | 항목 | 규칙 이름 또는 섹션 제목 |
| B | 내용 | 규칙 상세 설명 |

B열을 비워두고 A열에만 텍스트를 입력하면 **섹션 구분선**으로 표시됩니다.

---

## 사용법

1. 위 가이드에 따라 Google 스프레드시트를 작성합니다.
2. `index.html`을 브라우저에서 열면 스프레드시트 데이터가 자동으로 로드됩니다.
3. 스프레드시트 내용을 수정한 후 **페이지를 새로고침**하면 변경사항이 반영됩니다.

## 참고사항

- 별도 서버 없이 `index.html` 파일을 직접 열어서 사용할 수 있습니다.
- 데이터 로드에 실패하면 10초 후 에러 메시지가 표시됩니다.
- 스프레드시트가 **공개(뷰어)** 상태여야 데이터를 불러올 수 있습니다.
